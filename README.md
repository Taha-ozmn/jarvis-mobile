# J.A.R.V.I.S. Mobile

Stark OS tarzında basit bir görev yöneticisi (Expo / React Native).

## Gereksinimler

- [Node.js](https://nodejs.org/) 18+
- [Expo Go](https://expo.dev/go) (telefonda test için)

## Kurulum ve çalıştırma

```bash
git clone https://github.com/KULLANICI_ADIN/jarvis-mobile.git
cd jarvis-mobile
npm install
npx expo start
```

Telefonda Expo Go ile QR kodu tarayın veya `i` (iOS simülatör) / `a` (Android emülatör) tuşlarına basın.
